# Landing VidaPlena SpA

Este proyecto consiste en una landing page para VidaPlena SpA, una empresa dedicada al bienestar y la salud integral
El sitio permite mostrar sus beneficio , incluir testimonios de clientes y facilitar el contacto mediante un formulario.

## Descripción del proyecto
VidaPlena SpA necesitaba una pag web simple y atractiva para tener presencia digital y comunicar sus servicios a posibles nuevos clientes
La pagina cuenta con las siguientes secciones:

- Inicio
- Beneficios
- Testimonios
- Contacto
- Footer con información de contacto

## Tecnologías utilizadas
- HTML5
- CSS3
- Git
- GitHub

## Instrucciones de uso

1. Descargar el repositorio.
2. Abrir la carpeta del proyecto.
3. Ejecutar el archivo `index.html` en un navegador web.
4. Interactuar por las secciones Inicio, beneficios, testimonios y contacto.
